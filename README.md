## Проверяет знание количество дней в конкретном году

###Схема работы программы
1. Вывод сообщения в консоли "Введите год в формате "yyyy"". 
2. Ввод года в формате yyyy (например 2004).
3. Вывод сообщения в консоли "Введите количество дней" .
4. Ввод количества дней (например 365).
5. Проверка соответствия количества дней году. 
6. Повтор до тех пор пока пользователь не ошибётся; если пользователь ошибся, надо вывести правильный ответ. 
7. Вывод количества правильных вводов.
